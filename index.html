Calculadora con MVC






Modelo-Vista-Controlador (MVC)

En este proyecto, la arquitectura MVC se utiliza para organizar la lógica de la calculadora de manera estructurada y modular.

Modelo: El modelo gestiona los datos y la lógica de negocio. En este caso, maneja el nombre del usuario, los números involucrados en las operaciones, el resultado y la operación actual.

Vista: La vista se encarga de presentar la información al usuario. Aquí, actualiza el elemento `#resultado` con el resultado de la operación realizada.

Controlador: El controlador maneja las interacciones del usuario y actúa como intermediario entre la vista y el modelo. Asocia eventos a funciones específicas para actualizar el modelo y la vista.
